What is the difference between var and let?
    - can redeclare var, cannot redeclare let
    - var is function-scoped, let is block-scoped

What is the difference between var and const?
    var: can reassign, declare; is function scoped
    const: canNOT reassign, canNOT redeclare; is block scoped

What is the difference between let and const?
    let: can reassign
    const: canNOT reassign

What is hoisting?
    it allows functions to be safely used in code before they are declared
